Hello, Welcome to Couch Critique.

Couch Critique is a web-application that let's users write movie reviews, keep track of movies that they've seen, and subscribe to other users/movie genres for a personalize review/movie feed.

Functionality Breakdown by Entities

Movies

Create: The search bar in the home page will retrieve a list of movie data from an external API (https://www.themoviedb.org/). Once a user clicks on a movie result, that movie will be added to our database to keep track of our data.
Example Routes - /movies/search?q=cats&page=1

Read: On any movie page, data will be retrieved from the external API, as well as any data that we are storing in our database.
Example Routes - /movies/536869

User

Create: A user can be created on the register page.
Example Routes - /register

Read: User data is used in a variety of places, mainly the login page and the user home page, which displays all of user's followers, followings, watched movies, written movie reviews, custom feed, and subscribed genres.
Example Routes - /login , /user/home

Reviews

Create: A review can be created on any movie page.
Example Routes - /reviews/1

Read: Reviews are displayed in any movie page as well as a user's profile page.
Example Routes - /movies/536869, /user/home/reviews

Update: Reviews can be updated within the user's review page.
Example Routes - /reviews/edit/1

Delete: Reviews can be updated within the user's review page.
Example Routes - /user/home/reviews

Genres

Read: A list of a genres is displayed in the add/remove genres page, as well as in a user's subscribed genres' page.
Example Routes - /user/home/genres, /genres

Subscribed Genres

Read: All of a user's subscribed genres are displayed in the user's page.
Example Routes - /user/home/genres

Create: A subscribed genre relationship can be created in the add/remove genres page.
Example Routes - /genres

Delete: A subscribed genre relationship can be deleted in the add/remove genres page.
Example Routes - /genres

Followings

Create: A following relationship can be created on a user profile page, by clicking the follow button.
Example Routes - user/1

Read: All of a user's followers/followings will be displayed in their home page.
Example Routes - user/home/follow

Delete: A following relationship can be deleted on a user profile page, by clicking unfollow, or by deleting it in within the user's home page.
Example Routes - user/home/follow

Watched Movies

Create: A watched movie relationship can be created on any movie page, by clicking on the 'Add to Watched Movies' button.
Example Routes - /movies/536869

Read: All of a user's watched movie is displayed within their home page.
Example Routes - /user/home/watched

Delete: Any watched movie relationship can be removed from their corresponding pages by clicking the 'Remove from Watched Movies' button or removed within a user's home page.
Example Routes - /user/home/watched
